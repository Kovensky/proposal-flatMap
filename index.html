Array.prototype.flatMap & Array.prototype.flatten

Introduction#

Array.prototype.flatten returns a new array that is one dimensional with all sub-array elements concatted into it recursively.

Array.prototype.flatMap first maps each element using a mapping function, then flattens the result into a new array. It is identical to a map followed by a flatten, but flatMap is quite often useful and merging both into one method is slightly more efficient.

1Array.prototype.flatMap ( callbackfn [ , thisArg ] )#

When the flatMap method is called with one or two arguments, the following steps are taken:

  1. Let O be ? ToObject(this value).
  2. If IsCallable(callbackfn) is false, throw a TypeError exception.
  3. If thisArg was supplied, let T be thisArg; else let T be undefined.
  4. Let A be ? ArraySpeciesCreate(O, len).
  5. Perform ? FlattenIntoArray(A, O, 0, callbackFn, thisArg).
  6. Return A.

2Array.prototype.flatten()#

When the flatten method is called with no arguments, the following steps are taken:

  1. Let O be ? ToObject(this value).
  2. Let A be ? ArraySpeciesCreate(O, len).
  3. Perform ? FlattenIntoArray(A, O, 0).
  4. Return A.

3FlattenIntoArray(target, source, start [ , mapperFunction, thisArg ])#

  1. Let targetIndex be start.
  2. Let sourceIndex be 0.
  3. Let sourceLen be ? ToLength(? Get(source, "length")).
  4. Repeat, while sourceIndex < sourceLen
    1. Let P be ! ToString(sourceIndex).
    2. Let exists be ? HasProperty(source, P).
    3. If exists is true, then
      1. Let element be ? Get(source, P).
      2. If mapperFunction was provided, then
        1. Set element to ? Call(mapperFunction, thisArg , « element, sourceIndex, target »).
      3. Let spreadable be ? IsConcatSpreadable(element).
      4. If spreadable is true, then
        1. Let nextIndex be ? FlattenIntoArray(target, element, targetIndex).
        2. Set targetIndex to nextIndex.
      5. Else,
        1. If targetIndex≥253-1, throw a TypeError exception.
        2. Perform ? CreateDataPropertyOrThrow(target, ! ToString(targetIndex), element).
    4. Increase targetIndex by 1.
    5. Increase sourceIndex by 1.
  5. Return targetIndex.
